# Tinkering With Durable Objects

Some experiments with Cloudflare's Durable Objects

## Getting Started

```
yarn install
```

At all times have the type-checker running

```
yarn dev
```

In one terminal:

```
yarn server dev
```

In another

```
yarn site dev
yarn site storybook
```
